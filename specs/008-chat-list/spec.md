specify: Implement Chat List

Goal
Display user's chats. If the user accepts invite, there should be a list of friends that user can send messages to. 

Application has a chat list sorted by the time of the last message received or sent

Application allows user to archive and unarchive chats

Users are able to send text messages to each other

Features
- sorted by last message time
- tap to open chat
- archive chat
- unarchive chat

Acceptance Criteria
- chats sorted correctly
- archive works
- unarchive works

What to test
- Create multiple chats
- Send messages at different times
- Verify newest chat appears first
- Archive a chat
- Confirm it disappears from main list
- Unarchive chat
- Confirm it returns
