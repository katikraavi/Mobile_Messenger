specify: Implement Real-Time Messaging

Goal
Send and receive messages.
Users are able to send text messages to each other


Features
- text messages
- typing indicators
- message status: sent, delivered, read

User actions
- edit message
- delete message

Acceptance Criteria
- two users can chat
- typing indicator visible
- message statuses update

What to test
- Send message between two users
- Confirm message appears in receiver chat
- Start typing → verify typing indicator
- Check status progression: sent → delivered → read
- Edit a message
- Delete a message
