# Chat Invitations Feature - User Guide

## Overview

The Chat Invitations feature allows you to discover and connect with other users in the Mobile Messenger app. Send invitations to users you want to chat with, accept or decline incoming invitations, and manage your chat requests.

---

## How to Send an Invitation

### Step-by-Step

1. **Open the Invitations Tab**
   - Tap the "Invitations" tab at the bottom of the app

2. **Find the Send Button**
   - Look for the "Send New Invite" button at the top of the Invitations screen
   - Or tap the floating action button (if enabled)

3. **Search for a User**
   - In the search screen, type the username or name of the person you want to invite
   - Results will appear as you type

4. **Select a User**
   - Tap on the user you want to invite
   - The user will be highlighted

5. **Send Invitation**
   - Tap the "Send Invite" button
   - You'll see a confirmation message

6. **Success**
   - The invitation is sent!
   - The recipient will receive a push notification (if enabled)
   - You can view the invitation status in the "Sent Invitations" tab

### Restrictions

- **Cannot invite yourself** - You can't send an invitation to your own account
- **Already connected** - If you're already chatting with someone, you can't send another invitation
- **Already invited** - If you've already sent a pending invitation to someone, you can't send another until they respond

## Managing Incoming Invitations

### View Pending Invitations

1. Open the **Invitations** tab
2. Look at the **Pending Invitations** tab (default)
3. You'll see all invitations you've received from other users
4. Each invitation shows:
   - User's profile picture
   - User's name
   - When the invitation was sent
   - Accept and Decline buttons

### Accept an Invitation

1. Find the invitation in your Pending list
2. Tap the **Accept** button
3. A chat will be created between you and the inviter
4. You can now message them in the Chat screen
5. The invitation will be removed from Pending

### Decline an Invitation

1. Find the invitation in your Pending list
2. Tap the **Decline** button
3. The invitation will be rejected
4. The sender will know you declined (if they check sent invitations)
5. You can receive new invitations from this user later

## Viewing Sent Invitations

1. Open the **Invitations** tab
2. Switch to the **Sent Invitations** tab
3. You'll see all invitations you've sent
4. Each invitation shows:
   - Recipient's name
   - Current status: Pending, Accepted, or Declined
   - When you sent it

### Invitation Status

- **Pending** - Waiting for recipient to respond
- **Accepted** - Recipient accepted and chat is created
- **Declined** - Recipient declined your invitation

## Push Notifications

### Receiving Notifications

When someone sends you an invitation:

1. **iOS**: You'll see a notification banner at the top of your screen
2. **Android**: You'll see a notification in your notification drawer
3. **App Badge**: The Invitations tab will show a badge with the count of pending invitations

### Notification Content

The notification will show:
- Title: "Chat Invitation"
- Message: "New chat invitation from [User Name]"

### Tapping a Notification

If you tap a notification:
1. The app will open automatically
2. You'll be taken to the Invitations screen
3. The Pending tab will be active
4. You can immediately accept or decline

### Managing Notifications

To enable/disable notifications:
1. Go to the app's settings (or system settings)
2. Find the Notifications section
3. Toggle Invitations notifications on/off

## Offline Usage

### What Happens if I Go Offline?

Even if you lose internet connection, you can:
- View cached pending and sent invitations
- See the last known status of each invitation

### What's Not Available Offline

- Sending new invitations
- Accepting or declining invitations
- Refreshing the invitation list
- Receiving new notifications

### Auto-Sync When Online

When you regain connection:
1. The app will automatically fetch the latest invitations
2. Any actions you queued offline will retry
3. You'll see the most current data

## Troubleshooting

### "Cannot invite yourself" Error

This means you tried to send an invitation to your own account. Each user can only invite others, not themselves.

### "Pending invitation already exists" Error

You've already sent an invitation to this user that's still waiting for their response. Wait for them to accept or decline before sending another.

### "Users already have a chat" Error

You're already chatting with this person, so you don't need an invitation to message them.

### I Sent an Invitation but Don't See It in "Sent"

Try refreshing the screen by:
- Swiping down to refresh
- Switching tabs and back
- Closing and reopening the app

### I'm Not Receiving Notifications

Check:
1. App has notification permission (check in system settings)
2. Invitations notifications haven't been disabled
3. You're still logged in
4. Your device has an active internet connection

### The Invitation List Shows Stale Data

You're viewing cached data from earlier. When you go back online or refresh, the latest data will load.

## Best Practices

1. **Keep Names Recognizable** - Use your real name so users know who you are
2. **Add a Profile Picture** - People are more likely to accept invitations if they can see your face
3. **Be Patient** - Users might not respond immediately; they may be busy
4. **Don't Spam** - Avoid sending multiple invitations to the same person
5. **Use Search First** - Search for users before sending to make sure you have the right person

## FAQ

**Q: Can I cancel an invitation I sent?**  
A: Not directly. The recipient can accept or decline. Once accepted, a chat is created. Once declined, you can send a new invitation later.

**Q: What happens if I decline an invitation? Can I recover it?**  
A: When you decline, it's removed from your Pending list. However, the sender can send a new invitation later.

**Q: How many invitations can I send?**  
A: There's no limit, but users can only send one pending invitation per recipient at a time.

**Q: Do I need to accept an invitation to chat?**  
A: Yes, accepting the invitation is what creates the chat and allows messaging.

**Q: Can I see who's blocked me?**  
A: Not directly in Invitations, but if you try to send an invitation and get an error, you might be blocked.
